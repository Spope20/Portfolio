Mapping Earthquakes

"A image of the Earthquake mapping site focused on part of California."

This project features a website with an interactive map made using the mapbox API. The map features 2 buttons to control zoom, a hover menu button on the top right an a legend. From the menu the user can select their prefered view: Street, Satalite, and Dark. all the elements of the earthquake mapping can be toggled. the earthquakes are maped with circles that change size and color based on magnitude. A legend on the bottom right shows what range of magnitudes each color represents. When the user clicks on an earthquake a infobox will pop up showing magnitude and location
