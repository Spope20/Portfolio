World Weather Analysis

"Part of a trip planned along brazils coastline with matching arival and depature destination"

Skills: Python, API, matplotlib

This project was done to showcase using API calls in python. The primary APIs used were OpenWeatherMap and Google Maps. The OpenMaps API retrives weather data based on city name while the google maps api provides an interactive map to view the data with. The project maps out a trip to 4 cities chosen from a random list and displays their weather data. When using APIs, a config file with a key is required to use the project. API keys are private and thus not provided with the project. Images of the project output can be found in the folders with their respective scripts.
